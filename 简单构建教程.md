# FlashBrowser 超简单构建教程（无需代码基础）

## 方法一：使用GitHub自动构建（推荐⭐最简单）

这个方法只需要点几下鼠标，GitHub会自动帮你生成APK文件！

### 步骤1：注册GitHub账号
1. 打开 https://github.com
2. 点击 "Sign up" 注册账号（免费）
3. 验证邮箱

### 步骤2：创建代码仓库
1. 登录GitHub
2. 点击右上角 "+" 号 → "New repository"
3. 仓库名称填：`FlashBrowser`
4. 选择 "Public"（公开）
5. 点击 "Create repository"

### 步骤3：上传代码文件
1. 下载本项目代码（FlashBrowser文件夹里的所有文件）
2. 在GitHub仓库页面，点击 "uploading an existing file"
3. 把FlashBrowser文件夹里的所有文件拖进去
4. 点击 "Commit changes"

### 步骤4：开启自动构建
1. 在仓库页面，点击 "Actions" 标签
2. 点击 "I understand my workflows, go ahead and enable them"
3. 点击 "New workflow"
4. 选择 "set up a workflow yourself"
5. 把下面的内容复制进去：

```yaml
name: Build APK

on:
  push:
    branches: [ main ]
  workflow_dispatch:

jobs:
  build:
    runs-on: ubuntu-latest
    
    steps:
    - uses: actions/checkout@v3
    
    - name: Set up JDK 17
      uses: actions/setup-java@v3
      with:
        java-version: '17'
        distribution: 'temurin'
        
    - name: Setup Android SDK
      uses: android-actions/setup-android@v2
      
    - name: Grant execute permission
      run: chmod +x gradlew
      
    - name: Build Debug APK
      run: ./gradlew assembleDebug
      
    - name: Upload APK
      uses: actions/upload-artifact@v3
      with:
        name: FlashBrowser-APK
        path: app/build/outputs/apk/debug/app-debug.apk
```

6. 点击 "Start commit" → "Commit new file"

### 步骤5：获取APK文件
1. 等待几分钟（GitHub会自动构建）
2. 点击 "Actions" 标签
3. 点击最新的工作流运行记录
4. 滚动到底部，找到 "Artifacts" 部分
5. 点击 "FlashBrowser-APK" 下载
6. 解压下载的文件，里面就是 `app-debug.apk`

### 步骤6：安装到手机
1. 把APK文件传到手机（微信、QQ、数据线都可以）
2. 在手机上点击APK文件
3. 允许安装未知来源应用
4. 完成安装！

## 方法四：使用Docker一键构建（适合有一定电脑基础的用户）

### Windows用户

1. 下载并安装 Docker Desktop：https://www.docker.com/products/docker-desktop

2. 打开命令提示符（CMD），输入：
```batch
cd FlashBrowser文件夹的路径
```

3. 运行以下命令：
```batch
docker run --rm -v "%cd%":/project -w /project openjdk:17-slim bash -c "apt-get update && apt-get install -y android-sdk && ./gradlew assembleDebug"
```

4. 等待构建完成，APK会在 `app/build/outputs/apk/debug/` 文件夹里

### Mac用户

1. 安装Docker Desktop

2. 打开终端，输入：
```bash
cd FlashBrowser文件夹的路径
docker run --rm -v "$(pwd)":/project -w /project openjdk:17-slim bash -c "apt-get update && apt-get install -y android-sdk && ./gradlew assembleDebug"
```

## 常见问题

### Q: 构建失败了怎么办？
A: 尝试以下方法：
1. 检查是否所有文件都上传了
2. 重新运行构建（点击 "Re-run jobs"）
3. 检查错误信息，可能是网络问题

### Q: APK安装不了？
A: 
1. 确保手机Android版本 >= 10
2. 在设置中允许 "安装未知来源应用"
3. 关闭手机的 "纯净模式"（部分国产手机）

### Q: 如何更新APK？
A:
1. 修改代码（如果有需要）
2. 重新提交到GitHub
3. GitHub会自动重新构建
4. 下载新的APK

### Q: 如何修改应用名称或图标？
A:
1. 修改 `app/src/main/res/values/strings.xml` 里的 `app_name`
2. 替换 `app/src/main/res/drawable/` 里的图标文件
3. 重新提交构建
