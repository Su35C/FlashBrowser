# FlashBrowser 新手图文教程（完全不懂代码也能学会）

## 写在前面

这篇教程是专门为**完全不懂编程**的新手准备的。我会用最简单的语言，配上详细的步骤，让你轻松构建出自己的APK文件。

**不需要：**
- ❌ 懂代码
- ❌ 安装Android Studio
- ❌ 买任何东西

**只需要：**
- ✅ 一台电脑（Windows/Mac都可以）
- ✅ 网络连接
- ✅ 10分钟时间
- ✅ 一点点耐心

---

## 推荐方法：GitHub自动构建（最简单）

### 第一步：注册GitHub账号（2分钟）

1. 打开浏览器，访问 https://github.com
2. 点击绿色的 "Sign up" 按钮
3. 填写信息：
   - Email：你的邮箱
   - Password：设置密码（记住它！）
   - Username：用户名（英文，比如 zhangsan2024）
4. 点击 "Create account"
5. 去邮箱查看验证邮件，点击验证链接

> 💡 **小贴士**：GitHub是全世界最大的代码托管网站，免费使用！

---

### 第二步：创建仓库（2分钟）

1. 登录GitHub后，点击右上角 **+** 号
2. 选择 "New repository"（新建仓库）
3. 填写仓库信息：
   - Repository name：填 `FlashBrowser`
   - Description：可以填 "Flash游戏浏览器"
   - 选择 **Public**（公开，免费）
   - ✅ 勾选 "Add a README file"
4. 点击最下面的绿色按钮 "Create repository"

> 💡 **小贴士**："仓库"就像是你在GitHub上的一个文件夹，用来存放代码

---

### 第三步：上传代码文件（3分钟）

1. 在新建的仓库页面，点击 "Add file" 按钮
2. 选择 "Upload files"
3. 你会看到一个大的虚线框，写着 "Drag files here..."
4. **打开你的电脑文件夹**，找到FlashBrowser项目文件夹
5. **全选所有文件和文件夹**（Ctrl+A 或 Cmd+A）
6. **拖放到网页的虚线框里**
7. 等待文件上传完成（可能需要1-2分钟）
8. 滚动到页面底部，点击 "Commit changes"

> ⚠️ **注意**：一定要上传所有文件，包括隐藏的文件夹（如 .github）

**如果拖放不行，可以用这个方法：**
1. 先把FlashBrowser文件夹压缩成zip
2. 解压zip文件
3. 进入解压后的文件夹
4. 全选所有内容，拖放到GitHub

---

### 第四步：开启自动构建（1分钟）

1. 上传完成后，点击页面上方的 "Actions" 标签
2. 你可能会看到提示 "Workflows aren't being run on this forked repository"
3. 点击绿色的 "I understand my workflows, go ahead and enable them" 按钮
4. 等待几秒钟...

> 💡 **小贴士**：GitHub Actions是GitHub的自动构建服务，免费！

---

### 第五步：等待构建完成（5-10分钟）

1. 点击 "Actions" 标签
2. 你会看到一个正在运行的工作流（黄色圆点）
3. 点击这个工作流，可以看到详细的构建过程
4. **耐心等待**，不要关闭页面
5. 当黄色圆点变成绿色勾勾 ✅，说明构建成功了！

**构建过程包括：**
- 设置环境（1分钟）
- 下载依赖（2-3分钟）
- 编译代码（2-3分钟）
- 打包APK（1分钟）

> ⏰ **如果等了超过15分钟**：刷新页面，可能是卡住了

---

### 第六步：下载APK文件（1分钟）

构建成功后：

1. 在Actions页面，点击最新的成功构建记录
2. 滚动到页面最底部
3. 找到 "Artifacts" 部分
4. 你会看到两个文件：
   - `FlashBrowser-Debug-APK` （调试版本，推荐下载这个）
   - `FlashBrowser-Release-APK` （发布版本，未签名）
5. 点击 `FlashBrowser-Debug-APK` 下载
6. 下载的是一个zip文件，解压后得到 `app-debug.apk`

> 🎉 **恭喜你！这就是你的APK文件！**

---

### 第七步：安装到手机（2分钟）

#### 方法一：用微信/QQ传输（最简单）

1. 把 `app-debug.apk` 发送给自己的微信文件传输助手
2. 在手机上打开微信，下载文件
3. 点击下载好的APK文件
4. 手机可能会提示 "禁止安装未知来源应用"
5. 点击 "设置"，允许来自此来源的应用
6. 返回，再次点击安装

#### 方法二：用数据线

1. 用USB线连接手机和电脑
2. 在手机上选择 "文件传输" 模式
3. 把APK文件复制到手机的Download文件夹
4. 在手机上打开文件管理器
5. 找到Download文件夹，点击APK安装

#### 方法三：用网盘

1. 把APK上传到百度网盘/阿里云盘
2. 手机下载网盘APP
3. 下载APK文件
4. 点击安装

> ⚠️ **安装时可能遇到的问题**：
> - 提示 "解析包错误"：重新下载APK，可能文件损坏
> - 提示 "应用未安装"：先卸载旧版本
> - 提示 "不兼容"：检查手机Android版本是否>=10

## 备用方法：本地一键构建

如果你不想用GitHub，也可以在本地构建。

### Windows用户

#### 准备工作

1. **下载并安装JDK 17**
   - 访问：https://www.oracle.com/java/technologies/downloads/#java17
   - 下载 "Windows x64 Installer"
   - 双击安装，一路点击 "Next"

2. **验证Java安装**
   - 按 `Win+R`，输入 `cmd`，回车
   - 输入 `java -version`，回车
   - 如果显示版本号（如 17.0.x），说明安装成功

#### 开始构建

1. 打开FlashBrowser文件夹
2. 双击运行 `一键构建.bat`
3. 等待构建完成（5-10分钟）
4. 构建成功后，APK会在桌面上

### Mac用户

#### 准备工作

1. **安装Homebrew**（如果还没有）
   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```

2. **安装JDK 17**
   ```bash
   brew install openjdk@17
   ```

3. **验证Java安装**
   ```bash
   java -version
   ```

#### 开始构建

1. 打开终端
2. 进入FlashBrowser文件夹：
   ```bash
   cd FlashBrowser文件夹的路径
   ```
3. 运行构建脚本：
   ```bash
   ./一键构建.command
   ```
4. 等待构建完成
5. APK会在桌面上

## 常见问题解答

### Q1: GitHub构建失败了怎么办？

**可能的原因和解决方法：**

1. **文件上传不完整**
   - 重新上传所有文件
   - 确保 .github/workflows/build.yml 文件存在

2. **网络问题**
   - 点击 "Re-run jobs" 重新构建
   - 多试几次

3. **代码有错误**
   - 检查是否修改了任何代码文件
   - 如果有修改，恢复原始文件

### Q2: 如何更新APK到新版本？

1. 修改代码（如果需要）
2. 重新上传到GitHub
3. GitHub会自动重新构建
4. 下载新的APK

### Q3: 如何修改应用名称？

1. 在GitHub仓库中，找到 `app/src/main/res/values/strings.xml`
2. 点击编辑按钮（铅笔图标）
3. 找到 `<string name="app_name">FlashBrowser</string>`
4. 把 `FlashBrowser` 改成你想要的名字
5. 点击 "Commit changes"
6. 等待重新构建
7. 下载新的APK

### Q4: 如何修改应用图标？

1. 准备一张512x512像素的PNG图片
2. 在GitHub仓库中，找到 `app/src/main/res/drawable/ic_launcher_foreground.xml`
3. 删除这个文件
4. 上传你的PNG图片，重命名为 `ic_launcher_foreground.png`
5. 提交更改，等待构建

### Q5: 构建成功了但安装不了？

**检查以下几点：**

1. **Android版本**：手机必须是Android 10或更高版本
   - 查看方法：设置 → 关于手机 → Android版本

2. **允许安装未知来源应用**
   - 设置 → 安全 → 未知来源应用 → 允许

3. **关闭纯净模式**（华为手机等）
   - 设置 → 系统和更新 → 纯净模式 → 关闭

4. **卸载旧版本**
   - 如果之前安装过，先卸载再安装新版本

### Q6: 为什么有两个APK文件？

- **Debug版本** (`app-debug.apk`)：
  - 包含调试信息
  - 文件较大
  - 适合测试使用
  - ✅ 推荐下载这个

- **Release版本** (`app-release-unsigned.apk`)：
  - 没有调试信息
  - 文件较小
  - 未签名，需要自行签名才能安装
  - 适合正式发布

## 进阶：自定义你的APK

### 修改首页网址

1. 打开 `app/src/main/java/com/flashbrowser/MainActivity.kt`
2. 找到 `HOME_URL = "https://www.4399.com"`
3. 把网址改成你想要的
4. 提交到GitHub，等待构建

### 添加更多游戏网站支持

1. 打开 `app/src/main/java/com/flashbrowser/utils/UrlUtils.kt`
2. 找到 `isGameSite` 函数
3. 在列表中添加新的网站域名
4. 提交到GitHub，等待构建

### 修改主题颜色

1. 打开 `app/src/main/res/values/colors.xml`
2. 修改颜色值（使用十六进制颜色代码）
3. 提交到GitHub，等待构建
